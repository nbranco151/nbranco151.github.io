Getting data from the SQL database¶

The first thing to do is to connect to the database using a connection object. Generally, this is called "conn".

In [52]:
conn = sqlite3.connect('database.sqlite')

We also need a cursor object in order to be able to access the data.

In [53]:
c = conn.cursor()

We need the list of the tables in the database. To do that, we use the SELECT command to select all tables. SQL commands are usually written in all caps as a convention.

The command we just did returns a list of tuples, so we will have to work around that. To do so, I will be making a list and putting all the rows one by one into the list, then convert that list into an array.
